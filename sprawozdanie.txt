LAB 1: PODSTAWY GIT/GITHUB

a) Ćwiczenie 1 - Utworzenie repozytorium:
   - git init (już zainicjowane)
   - git branch -M main
   - git status

b) Ćwiczenie 2 - Pierwszy commit:
   - Poprawiono README.md na zwięzły format
   - Utworzono sprawozdanie.txt
   - git add .
   - git commit

Użyte komendy:
git status
git branch -M main
echo > README.md
echo > sprawozdanie.txt
